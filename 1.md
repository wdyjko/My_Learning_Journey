Flex 弹性布局入门：小白也能看懂

如果你还在用 float、position 做布局，可能会遇到对齐难、适配难的问题。Flex（Flexible Box）是为了解决这些痛点而生的布局方式，写法更简洁，效果更稳定。本文用最直白的方式讲清楚 Flex 的核心概念和常用属性，适合零基础同学快速上手。

一、Flex 是什么
开启 flex 布局的元素叫“容器”，它的直接子元素叫“项目”。只要给容器加上 display: flex，子元素就会按照 Flex 规则排布。

```html
<div class="box">
  <div class="item">A</div>
  <div class="item">B</div>
  <div class="item">C</div>
</div>
```

```css
.box {
  display: flex;
}
```

二、理解两根轴
Flex 里最重要的概念是两根轴：
1. 主轴：项目默认沿主轴排列。
2. 交叉轴：与主轴垂直的方向。

默认主轴是水平方向（从左到右），交叉轴是垂直方向（从上到下）。

三、容器常用属性

1）flex-direction：主轴方向
决定项目从哪边开始排。

```css
.box {
  display: flex;
  flex-direction: row;
}
```

可选值：
- row：从左到右（默认）
- row-reverse：从右到左
- column：从上到下
- column-reverse：从下到上

2）flex-wrap：是否换行
默认不换行，内容多了会被挤在一行里。

```css
.box {
  display: flex;
  flex-wrap: wrap;
}
```

可选值：
- nowrap：不换行（默认）
- wrap：换行
- wrap-reverse：反向换行

3）flex-flow：direction + wrap 的简写

```css
.box {
  display: flex;
  flex-flow: row wrap;
}
```

4）justify-content：主轴对齐方式
决定项目在主轴上的排列和空白分配。

```css
.box {
  display: flex;
  justify-content: space-between;
}
```

常用值：
- flex-start：靠主轴起点
- flex-end：靠主轴终点
- center：居中
- space-between：两端贴边，中间等距
- space-around：两侧留白相等
- space-evenly：所有间距相等

5）align-items：交叉轴对齐方式
单行项目在交叉轴上的对齐。

```css
.box {
  display: flex;
  align-items: center;
}
```

常用值：
- stretch：拉伸填满（默认）
- flex-start：靠上
- flex-end：靠下
- center：居中
- baseline：按文字基线对齐

6）align-content：多行整体对齐
只有在换行（flex-wrap: wrap）时才生效。

```css
.box {
  display: flex;
  flex-wrap: wrap;
  align-content: space-between;
}
```

四、项目常用属性

1）order：改变排列顺序
数字越小越靠前，默认是 0。

```css
.item {
  order: 1;
}
```

2）flex-grow：放大比例
有剩余空间时，项目按比例变大。

```css
.item {
  flex-grow: 1;
}
```

3）flex-shrink：缩小比例
空间不足时，项目按比例缩小，默认是 1。

```css
.item {
  flex-shrink: 1;
}
```

4）flex-basis：初始尺寸
在分配剩余空间前，项目的基础大小。

```css
.item {
  flex-basis: 200px;
}
```

5）flex：综合写法
最常用的简写形式。

```css
.item {
  flex: 1 1 200px;
}
```

记忆方式：flex = grow shrink basis。

6）align-self：单个项目对齐
覆盖容器的 align-items。

```css
.item {
  align-self: flex-end;
}
```

五、三个常见布局例子

1）水平垂直居中

```html
<div class="center">
  <div class="box">内容</div>
</div>
```

```css
.center {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 300px;
}
```

2）左右两列，左固定右自适应

```html
<div class="layout">
  <div class="left">左</div>
  <div class="right">右</div>
</div>
```

```css
.layout {
  display: flex;
}
.left {
  width: 200px;
}
.right {
  flex: 1;
}
```

3）导航均分

```html
<div class="nav">
  <div>首页</div>
  <div>课程</div>
  <div>关于</div>
</div>
```

```css
.nav {
  display: flex;
}
.nav > div {
  flex: 1;
  text-align: center;
}
```

六、常见误区与小贴士
1. align-content 只有在换行时有效，单行用 align-items。
2. 想让项目垂直居中，容器要有高度。
3. flex-basis 会覆盖 width 的计算优先级。
4. flex: 1 是最常见写法，代表可伸可缩，基础为 0。

七、快速记忆卡片
- 容器主轴：flex-direction
- 是否换行：flex-wrap
- 主轴对齐：justify-content
- 交叉轴对齐：align-items
- 多行对齐：align-content
- 项目顺序：order
- 项目伸缩：flex-grow / flex-shrink / flex-basis

总结
Flex 的核心就是“容器 + 两根轴 + 对齐方式”。掌握主轴与交叉轴后，再记住几个常用属性，你就能快速写出整洁、稳定的布局。建议把上面的三个例子亲手敲一遍，理解会更深。只要开始用，Flex 会成为你最常用的布局工具。
